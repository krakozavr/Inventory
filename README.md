## üöÄ Usage Instructions

### For StockPropsApp

1. **Import all files together**:
   - Select `index.html`, `styles.css`, `app.js`, and `data.js`
   - Import them simultaneously to StockPropsApp

2. **Files will be auto-inlined**:
   - CSS will be converted to `<style>` tags
   - JavaScript will be converted to inline `<script>` tags
   - Everything becomes a single self-contained HTML file

### For Testing Locally

1. Open `index.html` in any modern web browser
2. Enter a business name
3. Customize colors (optional)
4. Click "Continue" to proceed
5. Watch the loading animation
6. Explore the inventory system
